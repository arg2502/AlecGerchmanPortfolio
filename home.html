Syntripetal: 3D Unity Game

Syntripetal is a "rollie-pollie" platformer created in Unity 3D. The player rolls one of five balls through a set of levels designed to both entertain and challenge the player, while trying to collect items and beat the course quickly in order to get the highest score.

It was a five week long class project with a group of four people. Elements that I brought to the project were designing levels, composing music, and programming sound.

Chupacabra: 3D OpenGL Game

Chupacabra is a game created in OpenGL and programmed in C++ using Visual Studio 2013. The player plays as Carlos who is riding on his goat through a canyon when suddenly, very round chupacabras start chasing him. The player's goal is to throw objects at these very bouncy chupacabras to knock them off balance before they can get to Carlos.

This game was an eight week long project made with a group of four people. The game engine used, which rendered primitive objects and models onto a screen, was provided by the professor. Most of the work was divided amongst the group members. My main job was to make the chupacabras collide and bounce off all other objects.
